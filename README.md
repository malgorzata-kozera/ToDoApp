Here you can create your personalized tasks list.

## Technology Stack:
-	Python/Django
-	HTML5/CSS/Bootstrap4

## What can you do:
- register a new User
- add a new tasks
- mark them as an completed (nothing better than a view of all tasks done !)
- delete completed tasks
- delete all tasks
